🎬 CineVerse

Открой мир кино за секунды.

CineVerse — это современное веб-приложение для поиска фильмов и сериалов с использованием API The Movie Database (TMDB). Пользователи могут просматривать популярные фильмы, искать интересующие картины, фильтровать результаты по жанрам и получать подробную информацию о каждом фильме.

✨ Возможности
🔍 Поиск фильмов по названию
🎭 Фильтрация по жанрам
⭐ Просмотр рейтинга фильмов
📅 Отображение года выхода
📝 Подробное описание фильма
🎨 Современный адаптивный интерфейс
📱 Поддержка мобильных устройств

🚀 Запуск проекта
Клонируйте репозиторий:
git clone https://github.com/username/cineverse.git
Перейдите в папку проекта:
cd cineverse
Если используется серверная часть:

Создайте файл .env

TMDB_KEY=YOUR_API_KEY
Установите зависимости:
npm install
Запустите сервер:
npm start
Откройте браузер:
http://localhost:3000
📸 Функции приложения
Поиск фильмов

Находите фильмы и сериалы по названию.

Популярные фильмы

Просматривайте самые популярные фильмы на данный момент.

Жанры

Фильтруйте фильмы по категориям и жанрам.

Подробная информация

Получайте информацию о рейтинге, продолжительности, жанрах и описании фильма.

🔐 Безопасность

Для защиты API-ключа рекомендуется использовать серверную часть и хранить ключ в переменных окружения (.env). Не публикуйте секретные ключи в открытых репозиториях.
